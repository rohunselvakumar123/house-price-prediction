# House Price Prediction ML Project

## Overview

This project predicts house prices using Machine Learning algorithms built with Python and Scikit-learn.

The application allows users to input property details and receive estimated house prices using trained regression models.

---

## Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Machine Learning Regression Models
* Interactive Streamlit Web App
* Data Visualization using Matplotlib

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Streamlit

---

## ML Models Used

* Linear Regression
* Random Forest Regressor
* Gradient Boosting Regressor

---

## Project Structure

* app.py → Streamlit application
* dataset.xlsx → Dataset
* house_model.pkl → Trained ML model

---

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
